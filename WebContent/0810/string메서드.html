1. charAt / charCodeAt

str = "사자/호랑이/고양이/기린/하마/코끼리"
charAt(index) : index위치의 문자를 리턴
charCodeAt(index) : index위치의 unicode값을 리턴



2. indexOf / lastIndexOf

str = "가나다/사자/호랑이/고양이/기린/하마/코끼리/가나다"
indexOf("문자열") : 문자열의 위치를 검색
lastIndexOf("문자열") : 문자열의 위치를 뒤에서부터 검색



3. concat / trim

str2 = "딸기,복숭아,오렌지,수박,망고"
concat(): 문자열 합치기
str4 = " 사과 ,바나나,자두 "
trim() : 앞 뒤 공백제거



4. substring / slice / substr

str = "가나다/사자/호랑이/고양이/기린/하마/코끼리/가나다"
substring(start,end) : start~(end-1) 범위의 문자 추출
slice(start,end): start~(end-1) 범위의 문자 추출
substr(start,length): start 부터 length 개수 만큼 문자 추출



5. split

str = "가나다/사자/호랑이/고양이/기린/하마/코끼리/가나다"
split(구분자, 개수) : 구분자를 기준으로 잘라서 배열로 반환



6. replace

str = "가나다/사자/호랑이/고양이/기린/하마/코끼리/가나다"
replace(old, new) : old의 값을 new 값으로 변경
replace(//g) : /와 /사이에 정규표현식을 사용할 수 있다, g: global - old에 해당하는 값을 전부 찾아라
