[확인]버튼 클릭하면 proc1()함수가 호출된다
	proc1() 함수에서 Date객체 생성 후 
	document.write()를 브라우저에 출력한다.
	출력하기 전에 이미 만들어져있는 document는 write하여 새로쓰기때문에 사라진다.


	[출력]버튼 클릭하면 proc2()함수가 호출된다
	proc1() 함수에서 Date객체 생성 후 
	div에 출력한다.
	div에 출력하기 위해서는 접근방법이 필요하다.
	  방법 1 : document.querySelector('div'); - 만약 div가 여러개면 첫번째거 한개만 가져옴
	  방법 2 : document.querySelector('#out'); - 아이디 기술
	  방법 3 : document.getElementById('out');
	  방법 4 : document.getElementsByTagName('div');
	  
	div 안에 출력하는 방법
	  : div객체변수.innerHTML = 출력내용;
