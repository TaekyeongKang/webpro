span테그는 자동으로 줄이 바뀌지 않기 때문에 br테그로 줄 바꿔줘야한다.
지정된 영역 크기 만큼만 차지한다 : inline 요소 -자동으로 줄안바뀜.
가로,세로, 여백 크기 등을 지정할 수 없다.
단, 스타일에 display : inline-block; 을 설정하면 block요소의 특성을 사용할 수 있다.
앞 뒤 여백을 주기 위해서 스타일에 margin값을 부여한다.

img테그는 이미지를 나타내는 테그이다.
필수 속성으로 여는 테그에서 [src="이미지 이름"] 에 표시하고자 하는 이미지 이름을 설정한다.
이미지의 크기를 설정할 수 있다. : inline 요소 but 이미지자체의 크기를 설정




a테그는 페이지 이동을 실행하는 테그 : inline요소 필수속성으로 [href="페이지 이름"]을 가진다. 네이버
