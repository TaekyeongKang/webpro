1. Date객체 생성

- new Date()
- new Date(123456789) : 1970/01/01 로부터 인수만큼(ms) 경과한 날짜
- new Date("2010.02.23")
- new Date(2010,01,23,12,30,12)
- 월은 0월부터 시작, 숫자로 Date객체 생성시 하나 작은 값으로 설정



2. Date객체 생성

- now = new Date()
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간



3. 요일 구하기

- now = new Date()
- day = now.getDay() : 0(일요일)~6(토요일)



4. 경과시간(단위 : milliseconds) 구하기

- now = new Date()
- day = now.getDay() : 0(일요일)~6(토요일)
