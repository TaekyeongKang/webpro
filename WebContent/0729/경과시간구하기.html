1. 경과시간(단위 : milliseconds) 구하기

- 태어난 날로 부터 현재까지 경과한 날 수 구하기
- birday = new Date("태어난 날")
- today = new Date() : 현재 날짜
- 경과일 : birday.getTime()



2. 경과시간 구하기

- 오늘부터 1일 일 때 100일 후 구하기
- today = new Date()
- 100 일의 millisec 값 구하기
- today.getTime() +100 일의 millisec 값



3. 경과시간 구하기

- 오늘부터 크리스마스까지 남은날 구하기
- today = new Date()
- 크리스마스의 millisec 값 구하기
- 크리스마스.getTime() - today.getTime()
